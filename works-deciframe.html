Deciframe.com

The product

It is a responsive web site to search and filter for music chords.

Key words

Sphinx, Egypt, Music, Ludic

Concept

Have fun and share while learning

Background

Deciframe.com is a responsive web site created for people that are starting to play a new instrument and are learning the first notes. In the site, the user can search for songs that have the same musical chords they just learned. The search can find a new song in three different ways: By typing the name of a song they already know how to play and Decifra-me display song with the same chords; By selecting the chords that they have already learned and Decifra-me suggests the songs; And finally, by selecting a well known chords sequence.

Process

1 Context

According to the briefing, the web site should be called “Decifra-me” and have three different searching options: song name, chords and well known chords sequences.

2 Target

University students interested in music and learning a new instrument. Young people aged 18-35, that enjoy using technology in everyday life, like social interaction and enjoy playing songs among their friends.

3 Development

3.1 Concept

The name of the project “Decifra-me”, is translated as “decipher me” to english. Also “cifra” is a group of chords. The pun caused by the name choice, served as a guide to build the project concept. “Decipher me or I'll devour you” is a well known expression from the old sphinx enigma myth. Based on that, the group decided that the project would involve sphinx, music and humor.

3.2 Design

All project elements were designed bades com the hexagonal shape. Each six-sided geometric shape element fit each other side-by-side perfectly, as in a beehive. This was a way of optimizing space, considering that each section would display tenths of chords at a time.

3.3 Colors

We used a vibrant color scale, so that the user can easily distinguish each chord and to emphasize the playful theme.

3.4 Usability

Decifra-me is meant to be a game application, therefore each functionality is presented its own page, indicated by one the theme colors - pink, purple and blue.

Gamification tools - In order to engage the user, incentivizing them to learn even more chords and, consequently, learn new songs;

Feedback when the user presses a button or chord - So that the user feels good about pressing. The songs are presented as the user is search the chords - So that the user does not wait for the complete search to see the options.

Loading bar - To avoid that the user feels anxious. Musical style filter - To ease the search.

Final Result

Decifra-me is the kind of project that a find joy in working on, it related music to a fun theme, which gives room for other fun ideas to be implemented - the web site ads uses its mascot dressed up as iconic singers. From a visual point of view, it has attractive characteristics for me such as, flat design, vibrant complementary colors and regular geometric shapes.
